### SeBackupPrivilege

Since emily is in backup operator and has `SeBackupPrivilege` we can abuse it to dump ntds.dit database.

`SeBackupPrivilege` allows our user to bypass **Access Control List**.
We can retrieve the SAM (Security Account Manager) file that stores local user account information by abusing this privilege. Since the passwords in SAM hive are encrypted using the bootkey in SYSTEM hive, we will also retrieve that file.
